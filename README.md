## Installation

__Yarn__
```
yarn add @aeternity/[package name]
```

__NPM__
```
npm install @aeternity/[package name] --save
```

## Commands

__Building the plugin__
```
yarn run build
```

__Running Lints__
```
yarn run lint
```

__Running Tests__
```
yarn run test
```
